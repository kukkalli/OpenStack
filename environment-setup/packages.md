## OpenStack packages
### Enable the repository for Ubuntu Cloud Archive
```
# add-apt-repository cloud-archive:train
```

### Finalize the installation
```
# apt update && apt dist-upgrade
```

### Install the OpenStack client
```
# apt install python3-openstackclient
```
